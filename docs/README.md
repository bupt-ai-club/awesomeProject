# Awesome Project

平时总会看到一些有意思的AI相关项目，但每次看完就忘记了，后面想找的时候又找不到，没有一个地方来汇总，以便在后面想看的时候能随时找到。
为了解决这个问题，笔者将近期看到的AI相关的项目，在这里进行汇总，并且每周会在公众号做一次汇总和分享，而且在后续有时间的时候会对汇总的
项目做一些实践和解读，同时也欢迎感兴趣的同学一起加入~
