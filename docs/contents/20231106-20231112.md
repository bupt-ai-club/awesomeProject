- 链接：https://huggingface.co/spaces/ysharma/OpenAI_TTS_New


- 介绍：基于 OpenAI 的 TTS 模型将文本转换为逼真的语音音频，并提供了一个名为speech的 API。支持多种语言，比如英语、西班牙语、法语、德语、意大利语等，此外，还支持 tts-1、tts-1-hd 和 tts-2等多种语音模型，并且可以在5 种男声和2种女声中进行切换。
