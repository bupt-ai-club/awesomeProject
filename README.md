# Awesome Project


随着大模型的兴起，各类AI项目层出不穷，本项目以人工智能领域知识共享为目的，收集一些高质量的开源AI项目，帮助更多人了解相关东西，并且每周会在公众号做一次汇总和分享，而且在后续有时间的时候会对汇总的项目做一些实践和解读，同时也欢迎感兴趣的同学一起加入~


## 环境安装
### Node.js版本
Node v16

### 安装docsify
```shell
npm i docsify-cli -g
```


### 启动docsify
```shell
docsify serve ./docs
```
